Instructions

Game Setup

Select the number of user players (1-10) from the dropdown menu. For each user, enter their name and difficulty level when prompted.

Click 'Confirm Player' to move to the next player or 'Let's Play' when the last player details are entered.

Difficulty Level

  • Random - Car is assigned a random abilty
  • Easy - Car is assigned a very high abilty
  • Normal - Car is assigned a medium abilty
  • Hard - Car is assigned a very low abilty

By default each car is set to random difficulty level

Number Of Races

Choose the number of races in the game - maximum 10.

Game Speed

After each race is run, the finishing positions are revealed one by one starting with 10th place. The speed of this reveal is controlled by the Game Speed dropdown menu. The speed setting can be changed at any time during the game and the next race will be run at the new speed.

Boost Points

Boost points are awarded at the end of each race to improve the ability rating of your car.

  • 1st - 3rd = 2 points
  • 4th - 7th = 1 point
